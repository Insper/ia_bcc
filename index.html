Horário das aulas

Horário de atendimento

Quartas das 10:00 até 11:30.

Informações adicionais sobre os projetos e implementações

Todas os projetos e implementações irão utilizar a linguagem de programação Python. Algumas das implementações irão utilizar o toolkit Gym OpenAI e, talvez, outros toolkits.

Espera-se que os alunos desta disciplina tenham conhecimento de programação em Python.

As entregas dos projetos e exercícios serão via Github Classroom
